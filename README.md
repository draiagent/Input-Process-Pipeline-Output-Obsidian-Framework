# Input-Process-Pipeline-Output-Obsidian-Framework
IPPOO™ 智慧知識轉換框架是一套專為 AI 時代多模型協作設計的知識轉換框架，解決「多個 AI 工具各自獨立、無法共享記憶」的核心痛點。 透過五個層級——**輸入（Input）**粗整理、**處理（Process）**邏輯驗證、**管線（Pipeline）**自動化、**輸出（Output）**視覺化、Obsidian 記憶中樞——將 Gemini、Claude、Claude Code、GPT-5.6 等異質模型串聯成一條有紀律的知識生產線，並以本機端 Markdown 知識庫作為唯一事實來源，確保每一份產出都經得起「費曼複述、第一性原理、反例驗證」三重把關，而非單純的 AI 資訊搬運。 一句話定位： 讓 AI 幫你「快」，讓框架幫你「準」。
